## âœ¨ Core Features
* **Match Tracking**: Automatically tracks the latest DOTA2 matches of bound players in the group/channel and sends image-based reports afterward.
* **Rank Tracking**: Automatically tracks the rank information of bound players and sends notifications upon rank changes.
* **Data Queries**: Provides a series of commands like `player`, `match`, `hero`, and `item` queries. (Use `dota2guide` to get plugin help)
